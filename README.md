## 1. 구현한 기능 목록

## Show vs Present Modally

=> 첫 '다음' 버튼의 segue를 show로 하고, 두 번째 '다음' 버튼의 segue를 present Modally로 했는데 효과의 차이점을 모르겠다... 미션 다하고 알아봐야 겠다.
