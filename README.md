###Literal Expression

- **special literals**

| Literal | Type | value |
| ------- | ---- | ----- |
| `#file` | `String` | `#file` 코드가 실행된 파일의 이름입니다. | 
| `#line` | `Int` | `#line` 코드가 실행된 줄의 번호입니다. |
| `#function` | `String` | `#function` 코드가 실행된 함수의 이름입니다. |
| `#column` | `Int` | `#column` 코드가 시작되는 열의 번호입니다. |
