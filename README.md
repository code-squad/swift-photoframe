## 요구사항

- 사진액자 - ViewController 요구사항을 구현한 상태로 시작한다.
- [뷰컨트롤러 컨테이너 강의 슬라이드](http://public.codesquad.kr/jk/photoframe-ViewController-Containers.pdf)를 읽고 학습한다.
